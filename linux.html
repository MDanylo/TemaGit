Операційна система GNU/Linux

Коротка інформація про GNU/Linux

GNU/Linux — це операційна система на базі ядра Linux, яка складається з вільного програмного забезпечення. Її можна використовувати як для персонального, так і для серверного застосування. Однією з ключових особливостей GNU/Linux є її гнучкість, модульність та висока стабільність.

Найбільш популярні дистрибутиви GNU/Linux

  • Ubuntu: один із найбільш популярних дистрибутивів, відомий своєю простотою для початківців.
  • Debian: стабільний дистрибутив, на основі якого розроблені багато інших.
  • Fedora: сучасний дистрибутив, який випускає найсвіжіші новинки.
  • Arch Linux: дистрибутив для досвідчених користувачів, який надає повний контроль над системою.
  • CentOS: серверний дистрибутив на базі Red Hat.
  • Linux Mint: дистрибутив, орієнтований на зручність користування.

Приклади команд роботи в Linux CLI

Команда Опис
pwd Показує поточний робочий каталог.
echo Виводить текст або значення змінної в термінал.
ls Виводить список файлів у поточному каталозі.
cd Змінює поточний каталог.
touch Створює новий порожній файл.
mkdir Створює новий каталог.
cp Копіює файли або каталоги.
mv Переміщує або перейменовує файли/каталоги.
rm Видаляє файли або каталоги.
history Показує історію використаних команд.
